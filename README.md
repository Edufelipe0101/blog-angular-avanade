Projeto de blog pessoal desenvolvido com Angular 14 para compartilhar artigos sobre tecnologia.

## 📋 Pré-requisitos

- Node.js 16+
- Angular CLI 14
- Git

## 🛠️ Instalação

```bash
# Clone o repositório
git clone https://github.com/seuUsuario/angular-blog.git

# Acesse a pasta do projeto
cd angular-blog

# Instale as dependências
npm install
